# 🏴‍☠️ Pirate Booty Wallet - Built-in Miner Integration

## Overview
The Pirate Booty wallet includes a **built-in one-click miner** for instant mining adoption. Users can start mining PBT directly from their wallet without additional software.

---

## Features

### ✅ One-Click Mining
- Start mining with a single button press
- No technical knowledge required
- Automatic configuration
- Works on any hardware (CPU/GPU)

### ⚙️ Smart Configuration
- Auto-detect optimal thread count
- Background mining mode (mine when idle)
- Power-saving options
- Temperature monitoring

### 📊 Real-time Statistics
- Live hashrate display
- Blocks found counter
- Total rewards earned
- Mining efficiency metrics
- Estimated daily earnings

### 🎮 Gaming-Friendly
- Automatically pause when gaming
- Low priority mode
- Battery-aware (laptops)
- Thermal throttling

## Implementation Details

### Core Components

#### 1. Wallet Miner Class
```cpp
class wallet_miner {
public:
  // Control
  bool start(address, threads);
  bool stop();
  
  // Configuration
  void set_threads_count(size_t);
  void set_background_mode(bool);
  
  // Statistics
  mining_stats get_stats();
  
  // Events
  void set_block_found_callback(callback);
};
```

#### 2. Mining Thread
- Uses Pirate Hash (Cuckoo Cycle + CryptoNight)
- Automatic difficulty adjustment
- Efficient memory management
- Low CPU overhead when idle

#### 3. RPC Integration
```cpp
// Get block template from daemon
get_block_template() -> block_template

// Mine block with Pirate Hash
mine_block(template) -> block

// Submit if valid
submit_block(block) -> accepted/rejected
```

### Performance Optimization

#### Background Mining
```cpp
if (background_mode) {
  // Monitor CPU usage
  if (cpu_idle > 60%) {
    mine_full_speed();
  } else {
    mine_low_priority();
  }
}
```

#### Thermal Management
```cpp
if (cpu_temp > 80°C) {
  reduce_threads();
} else if (cpu_temp < 70°C) {
  restore_threads();
}
```

#### Battery Awareness
```cpp
if (on_battery && battery < 20%) {
  pause_mining();
  show_notification("Mining paused - Low battery");
}
```

## Technical Specifications

### Mining Algorithm
- **Phase 1:** Cuckoo Cycle (memory-hard)
- **Phase 2:** CryptoNight (CPU-friendly)
- **Combined:** Pirate Hash

### Performance Targets
- CPU (4-core): 100-500 H/s
- CPU (8-core): 500-1500 H/s
- Low overhead: <5% when background
- Fast startup: <2 seconds

### Resource Usage
- RAM: 2-4 GB
- CPU: Configurable (1-100%)
- Disk: Minimal (logs only)
- Network: <100 KB/s
